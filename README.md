# PharmaShield India AI

PharmaShield India AI is an India-specific memory-aware multi-agent RAG platform for pharmacy operations.
It helps pharmacy teams analyze:
- medicine availability
- substitution workflows
- MRP and affordability
- Jan Aushadhi options
- prescription compliance
- Schedule H/H1/X risk
- NSQ/spurious drug alerts
- online seller risk
- scheme/claim/reimbursement support

## Quickstart
```bash
cp .env.example .env
make up
```

- Frontend: http://localhost:3000
- Backend docs: http://localhost:8000/docs
