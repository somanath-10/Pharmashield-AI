# Safety Guardrails

The platform is pharmacist-support software, not a medical decision system.

Guardrails enforced in the MVP:

- Never present outputs as final medical decisions.
- Never recommend dose changes without pharmacist or clinician review.
- Never tell a patient to use an unverified online supplier.
- Never treat public recall or enforcement records as proof of causation beyond the cited source content.
- Always state that pharmacist review is required.
- Label case-data-only conclusions as based on provided case data.
- Escalate suspicious GLP-1 authenticity signals to verification steps before dispensing or patient use.
- Generate prior authorization drafts and documentation requests, not final legal determinations.
